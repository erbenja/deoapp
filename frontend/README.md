# deoapp-frontend

## Project setup
### Required
```
 npm version ^6.2.0 
```
### Prepare dist

Replace baseURL in http-commons.js with address of backend
```
 npm install
 npm build
```
**Folder dist has all needed content ready to be hosted on web server.**
