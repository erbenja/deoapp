# deoapp-frontend

## Project setup
Required
```
 npm version ^6.2.0 
 postgresql database version ^12.2 
```
Prepare dist

Create ormconfig.js in root folder and fill in correct credentials (ormconfig-example.js)
```
 npm install
 npm run start
```
Folder dist has all needed content ready to be hosted on web server.
